# **Readme**

**Requirements**

- python 3.8.0
- virtualenv 20.6.0
- pip 19.2.3

**Installation**


1. create virtual environment 'venv':
   
   `virtualenv venv`

2. Load virtual python environment

   `source venv/bin/activate`

2. Install dependencies:
   
    `pip install -r requirements.txt`

3. Run

   `python -m src.main`
